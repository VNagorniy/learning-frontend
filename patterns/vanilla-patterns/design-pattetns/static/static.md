# Static паттерн

Это **import** ключевое слово позволяет импортировать код, экспортированный другим модулем. По умолчанию все статически импортируемые модули добавляются в исходный пакет.

Модуль, импортируемый с использованием стандартного синтаксиса импорта ES2015, _import module from 'module'_, импортируется статически.

Модули выполняются, как только движок достигает строки, на которой мы их импортируем.

## Пример использования

Поскольку компоненты были статически импортированы, Webpack объединил модули в исходный пакет.

Исходный код нашего чат-приложения упакован в один пакет: **main.bundle.js**. Большой размер пакета может существенно повлиять на время загрузки приложения в зависимости от устройства пользователя и сетевого подключения.

[Пример реализации](https://codesandbox.io/embed/pedantic-keldysh-bv7cj).

Вместо того, чтобы импортировать все компоненты статически, мы можем динамически импортировать модули после того, как Appкомпонент отобразит своё содержимое, и пользователь сможет взаимодействовать с нашим приложением.
